This is version 1.1 of ispectrum written by Charles Carrano, Boston College.

Included are precompiled binaries for Linux (ispectrum) and Windows (ispectrum.exe).

To compile the software under Linux

   make -f Makefile.linux

To compile the software under Windows, download Pelles C (http://www.smorgasbordet.com/pellesc/) and open the project file ispectrum.ppj

Run ispectrum with no command line arguments for additional help.
